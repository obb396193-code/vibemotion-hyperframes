# Open Design 设计规范富矿（扒自 nexu-io/open-design）

> 这是从 open-design 的 150 个 `DESIGN.md/SKILL.md` 设计系统里**挑科普/HF 相关的 18 个**扒来的（纯文本，236K）。
> **定位 = 设计规范「来源」，不是验证过的主题。** 想拿它当 vibemotion 新主题/界面规范 → **先渲染一条片验证**（铁律：没在 HF 动画里验证过的审美不进主题库）。当**色板/版式/字体的参考富矿**用最稳。
> ⚠️ **这只是 18 个，open-design 还有 ~100 个**（zhangzara 系列编辑/海报风一大把等）——缺了去 `nexu-io/open-design/design-templates` 挖，别只用这 18 个。

## 分类
### ⭐ HF / 视频专用（最相关，可直接学）
| 模板 | 一句话 |
|---|---|
| `hyperframes` | 官方 HF 设计模板：composition 契约 + scaffold 快路径(`npx hyperframes init --example blank`) + vfx 触发(liquid-glass/iphone/portal) |
| `motion-frames` | 动态帧/MG 规范 |
| `video-shortform` | **短视频规范**（竖屏/卡点向，对你最对口） |
| `sprite-animation` | 精灵帧动画（呼应 pixel 主题） |

### 讲解 / 知识向 PPT（科普选题富矿）
| 模板 | 适合做啥科普 |
|---|---|
| `html-ppt-tech-sharing` | 技术分享/讲解 |
| `html-ppt-knowledge-arch-blueprint` | 知识架构蓝图（讲体系/框架） |
| `html-ppt-course-module` | 课程模块（教学向） |
| `html-ppt-presenter-mode-reveal` | 演示揭示节奏 |
| `html-ppt-graphify-dark-graph` | 暗底图表/数据可视化 |
| `html-ppt-taste-editorial` · `html-ppt-zhangzara-soft-editorial` | 编辑/杂志风（沉静讲解） |
| `html-ppt-obsidian-claude-gradient` | **Claude 渐变风**（和我们 warm/Claude 主题呼应，重点看） |
| `guizang-ppt` | 一套完整 PPT 设计系统（18K，最详细） |

### 文档 / 长文 / 点评向
| 模板 | 用途 |
|---|---|
| `docs-page` · `blog-post` · `digital-eguide` | 文档/博客/电子指南版式 |
| `critique` | 点评/评审版式（9.6K，详细） |
| `live-artifact` | 动态 artifact 规范 |

## 怎么用
1. 做某科普片想要某种"调"→ 翻对应模板的 `SKILL.md` 拿**色板/字体/版式/组件结构**。
2. 看 `example.html`（部分模板有）感受成品。
3. 觉得某套适合做成 vibemotion 主题 → **先按它做一条片渲染验证**，好看再写进 `references/theme-*.md` 入库。
4. 不够 → 去 open-design 仓库挖剩下 ~100 个。
